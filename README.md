## Instructions
After cloning into repo, cd to project root directory and install dependencies:

```
$ npm install
```

To run dev server, install Nodemon library globally:

```
$ sudo npm install nodemon -g
```

Then run devserver from project root directory:

```
$ nodemon
```

Open browser to:

```
http://localhost:3000
```

Profit!
